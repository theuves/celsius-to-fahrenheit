# celsius-to-fahrenheit

> Convert celsius to fahrenheit.

## Installation

```
$ npm install --save celsius-to-fahrenheit
```

## Example

```js
var celsiusToFahrenheit = require('celsius-to-fahrenheit');

celsiusToFahrenheit(19);
//=> 66.2
```
